# Nutrition Daemon

栄養管理アプリケーション

## 概要

このプロジェクトは、ゲームを楽しみながら栄養学を学べるアプリケーションを目指しています（絶賛作成中）。

## セットアップ

### 前提条件

- Node.js 18.0.0以上
- npm または yarn

### インストール

```bash
# 依存関係のインストール
npm install

# 開発モードで起動
npm run dev

# 本番モードで起動
npm start
```

## 使用方法

詳細な使用方法については、各機能のドキュメントを参照してください。
